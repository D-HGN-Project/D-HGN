# Data Directory

Place the processed ADNI rs-fMRI and DTI files here before running the experiments.

Expected structure:

```text
data/
├── CN.csv
├── EMCI.csv
├── AD.csv
├── CN/GretnaDFCMatrixZ/
├── CN/Final_SC_Matrices/
├── EMCI/GretnaDFCMatrixZ/
├── EMCI/Final_SC_Matrices/
├── AD/GretnaDFCMatrixZ/
└── AD/Final_SC_Matrices/
```

ADNI data are not redistributed with this repository.
